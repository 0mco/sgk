# sgk
仿findmima的社工库代码，基于elasticsearch和PHP构建

#优点

1.基于elasticsearch，在数据的处理上要由于spinx</br>
2.比solr简单</br>
3.数据清洗，格式化和备份都比solr和spinx要简单</br>
4.速度快啊</br>

#缺点

1.前台查询语句还不够好</br>
2.界面直接用的findmima的，有些样式还不够完美</br>
3.希望UI/前端大牛能bugfix</br>
